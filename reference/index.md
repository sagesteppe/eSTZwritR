# Package index

## Main Functions

User facing functions

- [`regionCoding()`](https://sagesteppe.github.io/eSTZwritR/reference/regionCoding.md)
  : Determine which interior regions the STZ should be marked with
- [`dirmakR()`](https://sagesteppe.github.io/eSTZwritR/reference/dirmakR.md)
  : Create a directory and write out all empirical stz data
- [`mapmakR()`](https://sagesteppe.github.io/eSTZwritR/reference/mapmakR.md)
  : Create standardized maps for empirical seed transfer zones
- [`fieldsmakR()`](https://sagesteppe.github.io/eSTZwritR/reference/fieldsmakR.md)
  : Ensure that all fields in a shapefile are named correctly, and in
  the right order.
- [`orderZones()`](https://sagesteppe.github.io/eSTZwritR/reference/orderZones.md)
  : Suggest seed zones names, or at least numeric orders, based on input
  polygons

## Data

Data sets for examples and functions

- [`countries`](https://sagesteppe.github.io/eSTZwritR/reference/countries.md)
  : Borders of three North American nations
- [`omernik`](https://sagesteppe.github.io/eSTZwritR/reference/omernik.md)
  : Omernik level 3 ecoregions of the USA
- [`regions`](https://sagesteppe.github.io/eSTZwritR/reference/regions.md)
  : Unified interior region boundaries
- [`Carto_cities`](https://sagesteppe.github.io/eSTZwritR/reference/Carto_cities.md)
  : Cities for cartography
- [`acth7`](https://sagesteppe.github.io/eSTZwritR/reference/acth7.md) :
  An empirical seed transfer zone
- [`GAI-cont`](https://sagesteppe.github.io/eSTZwritR/reference/GAI-cont.md)
  : A raster of Global Aridity Index version 3
- [`GAI-coarse`](https://sagesteppe.github.io/eSTZwritR/reference/GAI-coarse.md)
  : A raster of Global Aridity Index version 3
